# 🇦🇺 Australian 417 Visa Postcode Map

Interactive map of Australian postcodes with **417 Working Holiday Visa** eligibility overlay.

## Features
- Colour by State or by Remoteness (ABS classification)  
- **Agricultural & Remote Zones** toggle (green) — Regional, Remote & Very Remote, Northern Australia  
- **Emergency Declared Areas** toggle (red) — Natural Disaster and Bushfire Declared Areas  
- Click any postcode for details incl. 417 eligibility and permitted work types  
- Hover-highlight with 280ms delay; fitBounds zoom on click

## Live Map
👉 **[Open the map](https://claudeforlani.github.io/australian-417-visa-map/)**

## Data Sources
- Postcode boundaries: ABS 2021 Postal Area boundaries  
- 417 eligible postcodes: [DHA Specified Work page](https://immi.homeaffairs.gov.au/visas/getting-a-visa/visa-listing/work-holiday-417/specified-work)  
- Remoteness classification: ABS ARIA 2016
